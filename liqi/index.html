介绍

昵称 秋风木叶，或 Yang。🤓


你都在使用哪些硬件？

  • iPhone 6s Plus (Nana)
  • MacBook Pro (Retina, 13-inch, Early 2015) (Anna)


软件呢？

我喜欢精简，系统自带的软件足够好则优先使用。

在满足自己所需要功能的前提下，将自己的互联网帐号降到最低，涉及到的平台和APP减到最少。

macOS Mojave (10.14)

  1. 神器: 终端 (Terminal.app)

    • Xcode命令行工具 xcode-select --install

    • 其他

      • JDK
      • MySQL
      • Node.js
        • npm config set registry https://registry.npm.taobao.org
        • sudo chown -R $(whoami) /usr/local/lib/node_modules
        • npm list -g --depth 0
      • jdk1.6

    • 终端代理

      • export ALL_PROXY=http://127.0.0.1:1087
      • unset ALL_PROXY

    • oh-my-zsh

      • zsh-syntax-highlighting
        • git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
      • zsh-autosuggestions
        • git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
      • zsh-completions
        • git clone https://github.com/zsh-users/zsh-completions ~/.oh-my-zsh/custom/plugins/zsh-completions
      • zsh-history-substring-search
        • git clone https://github.com/zsh-users/zsh-history-substring-search ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-history-substring-search

    • Homebrew (/usr/local/Cellar/)

      • Python3
        • pip3 install requests
        • pip3 install bs4
      • wget
      • tldr
      • tree
      • ffmpeg FFmpeg笔记

  2. 浏览器

    • 主要 Safari [Safari Extensions (Safari 13 之后废弃，强制转向AppStore)]

      • AdGuard for Safari (很卡，但效果最好) [App Store]
      • Tampermonkey [App Store]
        • 哔哩哔哩（bilibili.com）播放器调整R
        • bilibili merged flv+mp4+ass+enhance
        • Download Youtube videos and subtitles

    • 备用 Chrome

      • uBlock Origin
