今天看到一个有趣的项目 利器社群计划 ，借此机会，我也分享一下自己使用的一些小工具……

Update: 2018-06


介绍

昵称 Yang。崇尚完美，向往自由。🤓


你都在使用哪些硬件？

  • MacBook Pro 2015 (Anna)
  • iPhone 6sp (Nana)


软件呢？

我喜欢精简，系统自带的软件足够好则优先使用。

在满足自己所需要功能的前提下，将自己的互联网帐号降到最低，涉及到的平台和APP减到最少。

macOS 10.13.5

  1. 神器: Terminal (Mac自带的终端)

    • Xcode命令行工具 xcode-select --install

    • 其他

      • JDK
      • jdk1.6
      • MySQL
      • Tomcat
      • Node.js

    • oh-my-zsh

      • zsh-autosuggestions
      • zsh-syntax-highlighting
      • 终端代理
        • export http_proxy=http://127.0.0.1:1087;export https_proxy=http://127.0.0.1:1087;
        • export ALL_PROXY=socks5://127.0.0.1:1086
        • unset ALL_PROXY

    • Homebrew

      • 权限 chown -R $(whoami) /usr/local (zsh)
      • 位置 /usr/local/Cellar/
      • python3
      • ffmpeg
        • 简单剪切: ffmpeg -ss 0:0:0(起点时间) -t 00:10:00(持续时间) -i in.mp4 -c copy /Users/yang/Desktop/out.mp4
      • tree
      • mycli
      • youtube-dl
      • you-get
      • 找回macOS10.13的FTP （也可以从High Sierra之前的Mac把ftp和telnet拷到你电脑里，/usr/local/bin）
        • brew install telnet
        • brew install inetutils
        • brew link --overwrite inetutils

  2. 浏览器

    • 主要 Safari

      • uBlock Origin [Safari Extensions]
      • Evernote Web Clipper [Safari Extensions]
      • Tampermonkey

    • 备用

      • Chrome
      • Firefox


  1. 编辑器:

    • Atom

    • VSCode

      (大多数情况下，利用Mac的分屏功能，将终端放在左边，编辑器放在右边，可以完成大多数写代码的需求)
